1.功能&输出说明
    运行后输出2个图集
    蓝色条的图集为“环境预测”，蓝条长度代表游戏环境中不同英雄/阵容的占比的多少，从4个图的演变可以大致估计未来环境中不同忍者/阵容数量的变化趋势（假设玩家都追求胜率）
    绿色条的图集为“平衡性评估”，体现在当前/未来的游戏环境中，不同阵容达到预期胜率所需要的使用占比（比较难理解），可以简单理解为绿条越长的阵容越符合当下版本可以无脑选，长绿条阵容较少或出现有起点不是0的绿条则平衡性有问题

2.运行exe版本注意
    最好打开一个cmd窗口，然后把exe拖进去运行，这样运行结束后不会自动关闭窗口从而看到中间信息

3.运行Python脚本的环境要求（使用exe版本无要求）
    需要python和一些库环境，可选安装专用编辑器（比如sublime text3）
    python：
        https://www.python.org/downloads/windows/
        https://zhuanlan.zhihu.com/p/32668728
        安装完成后需要将exe路径添加到系统环境变量
    库：
        pip（安装其他python库的工具，后面的库都通过它安装）：
            https://pypi.org/project/pip/
            https://www.runoob.com/w3cnote/python-pip-install-usage.html
            安装完成后需要将exe路径添加到系统环境变量
        pandas、seaborn、numpy、pulp
            都用pip安装，cmd下输入pip install [库名]
            
4.数据文件要求
	2个csv文件分别存放对局胜率矩阵和对局场次矩阵，2个文件第一行、第一列都是忍者/阵容名，且内容和顺序需相同
    无数据的格子会被自动识别为0
    对局胜率举证还需要额外的2列“使用下限”和"使用上限"
